*************************
installation requirements
*************************
1. pip install requirements.txt
2. npm install vite
3. npm install lodash.debounce


*********************
Firing up the Web App!
**********************

Ports List:
5000, 5173

1. cd into Frontend
run "npm run dev"

2. cd into Backend
run "Python Flask_App.py"

If you error out then try to run:
rm -rf node_modules package-lock.json
npm install
